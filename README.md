# blockchainv2


Simple blockchain http and tcp server.

## How to run?

#### Install it

``` go get github.com/minhajuddinkhan/blockchainv2/server/blockchainv2 ```

#### Start the server 

``` go run blockchainv2 ``` 


#### Ping via nc/curl

#### Through nc

``` nc localhost 3000 ```

#### Through cURL

``` curl http://localhost:8080 ```
